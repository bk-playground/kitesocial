## Welcome

Welcome to Kitesocial, a twitter clone!

## Development setup

To run Kitesocial, you will need:

- Ruby 2.6.5
- Rails 6.0.2
- Yarn 1.19.1

Once you've cloned the repo:

```
  bundle install
  rails db:migrate RAILS_ENV=development
  rake db:seed
```

Then you'll be able to spin up your server!
